\# Smart Event Experience System



\## Problem

Large sporting events face:

\- Crowd congestion

\- Long queues

\- Poor real-time communication



\## Solution

A cloud-based system that:

\- Monitors crowd density

\- Provides real-time updates

\- Reduces waiting time



\## Tech Stack

\- Google Cloud Run

\- Firestore

\- Pub/Sub

\- Flask (Backend)

\- HTML/JS (Frontend)



\## Features

\- Live crowd status

\- Smart gate suggestions

\- Real-time updates



\## How to Run

Backend:

python main.py



Frontend:

Open index.html
